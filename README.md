# Empório dos Óculos — Site Institucional
Site desenvolvido como projeto de portfólio por Diogo Falcão (FalAiquoc)

🏪 Sobre o Negócio
A Ótica Empório dos Óculos é especialista em saúde visual no bairro de Lagoa Nova em Natal/RN. Oferecemos as melhores marcas de armações, lentes de alta tecnologia e atendimento especializado com preços de atacado direto de fábrica.

🛠️ Tecnologias
- Vite + React + TypeScript
- Tailwind CSS
- Lucide React

📐 Referências de Design
- https://github.com/adrianhajdin/project_hoobank — Referência para layout moderno e modular com Tailwind CSS e React.
- https://github.com/tailwindlabs/tailwindui-templates — Boas práticas de interface responsiva e espaçamento premium.
